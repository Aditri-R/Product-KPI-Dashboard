# 📊 Product Performance Dashboard UI

## Overview

A modern analytics dashboard designed in Figma for product managers and business teams to monitor key business metrics in one place. The dashboard emphasizes readability, visual hierarchy, and quick decision-making.

---

## Objectives

- Present important KPIs at a glance
- Improve data visualization
- Reduce dashboard clutter
- Create an intuitive analytics experience

---

## Features

- Revenue & Sales Analytics
- Customer Lifetime Value
- Product Engagement Metrics
- Acquisition & Conversion KPIs
- Retention & Loyalty Tracking
- Product Quality Monitoring
- Interactive Filters
- Clean Card-based Layout

---

## Tools Used

- Figma
- Auto Layout
- Components
- Variants
- Constraints
